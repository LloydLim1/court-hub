# CourtHub Mobile

An Expo Router + TypeScript scaffold for the CourtHub mobile experience.

## Included screens

- Home
- Login
- Signup
- User dashboard
- Admin dashboard

## Start

```bash
npm install
npm run start
```

## Notes

- The app is intentionally self-contained under `mobile/app`.
- Styling leans into a polished blue sports-club feel with stacked cards instead of grid-heavy layouts.
- Routes are scaffolded for navigation preview and are ready for real auth and data wiring.
